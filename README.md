# Patitas Felices

Interfaz web y movil para una fundacion de mascotas 

## Descripcion general del proyecto

Este proyecto consta de la creacion del diseño de una pagina web responsive usando HTML y CSS; de acuerdo a los siguientes requerimientos: 

**Página de inicio (Home)**
Banner principal:
Nombre de la fundación, logo o marca textual y un slogan claro.
Botón de llamada a la acción (ej. “Conócenos” o “Adopta”).
Diseño atractivo y prioridad visual.

**Carrusel de fotografías**
Muestra imágenes de animales/voluntariado/actividades.
Cambia automáticamente cada 3 segundos mediante CSS (@keyframes), sin uso de JavaScript.
Texto superpuesto opcional (breve caption).

**Secciones destacadas**
Resumen de la labor (qué hacen).
Botones/links internos a “Adopta”, “Donaciones”, “Voluntariado” y “Contacto”.
Testimonios (opcional): citas breves de adoptantes o voluntarios.

**Visión y Misión**
Deben aparecer lateralmente (lado a lado) en pantallas grandes y apilarse en mobile.
Cada bloque con título, icono o imagen pequeña y texto explicativo.
Diseño armonizado: iconografía y espaciado coherente.
Galería / Carrusel secundario
Mosaico o carrusel simulado (CSS) con fotos de rescates y eventos.
Debe ser responsive y mantener proporciones.
Animales disponibles / Adoptar (Catálogo simple)
Grid de tarjetas con animales (mínimo 8 ejemplos representativos).
Cada tarjeta muestra: imagen, nombre, edad aproximada, tamaño/raza, estado (available/adopted) y botón “Ver perfil”.
No se requiere lógica de filtro ni backend; basta maqueta estática que muestre distintos estados.

**Perfil de animal (Detalle estático)**
Vista ampliada con foto grande, descripción (historia, salud, comportamiento), características (edad, sexo, tamaño), y un call-to-action “Solicitar adopción” (no funcional).
Simular sección de “animales relacionados” en carrusel horizontal si se desea.

**Formulario de contacto**
Campos obligatorios (marcados visualmente):
Nombre.
Apellidos.
Ciudad.
Correo electrónico.
Teléfono (opcional o requerido según criterio).
Tipo de consulta / envío (select): Adopción / Donación / Voluntariado / Información general / Otro.
Mensaje**
